#Librarium

Librarium é um ecossistema de simulação de uma
biblioteca utilizando uma arquitetura de microsserviços.

##Booking

A API Booking é responsável pela entrada e saída de livros.

Funcionalidades:
- Apresentar livros
- Apresentar livros por categoria
- Login
- Logout
- Alugar livro
- Devolver livro

Modelos

- Livro:
    - Título
    - Resumo
    - Autor
    - Data lançamento

- Estoque:
  - Livro
  - Quantidade

- Usuario:
  - Nome
  - Sobrenome
  - Email
  - Senha
